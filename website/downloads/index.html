Distribution

i3 is currently distributed in the following operating systems/distributions:

  • Debian GNU/Linux
  • Arch Linux, maintained by Atsutane
  • Gentoo Linux (in sunrise), maintained by mimi_vx / Moredread
    See also: i3-9999.ebuild
  • FreeBSD (x11-wm/i3), maintained by dhn
  • NetBSD (wip/i3 in pkgsrc-wip)
  • OpenBSD, maintained by bapt
  • Fedora, RHEL, CentOS (in EPEL), maintained by cassmodiah
  • Ubuntu Linux (in karmic/universe)
    Because Ubuntu packages are always outdated, you can also install up-to-date third-party i3 packages for Ubuntu by adding the following line to your /etc/apt/sources.list and using apt-get install i3 afterwards:
        deb http://debian.sur5r.net/i3/ karmic universe
    Or, if you are running lucid: 
        deb http://debian.sur5r.net/i3/ lucid universe

Development version

Use git to follow the latest changes: 

  $ git clone git://code.stapelberg.de/i3

If you prefer to download a tarball, or if you cannot use git for whatever reason, you may download the current master branch from http://code.stapelberg.de/git/i3/snapshot/i3-master.tar.bz2
